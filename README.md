# 💊 DrugManage - お薬管理アプリ

DrugManageは、薬の飲み忘れを防止するためのAndroid向け管理・通知アプリです。
Jetpack Composeを用いて構築されたモダンなUIと、柔軟なスケジュール管理機能を備えています。

## ✨ 主な機能

- **柔軟な服薬スケジュール**：「毎日」「1日空け（2日ごと）」「N日置き」など、薬ごとに細かく間隔を設定できます。
- **スケジュールの起点設定**：「何日前に飲んだか」を指定することで、過去の服薬履歴を基準にした正確な次回予定を算出します。
- **フライング服薬対応**：予定時刻の前に「飲んだ」場合でも、直近のスケジュールを自動でスキップし、正確な次回スケジュールへと更新します。
- **カスタム通知**：時間通りだけでなく「10分前」「1時間前」など、薬ごとに通知タイミングを選べます。
- **カレンダー・履歴管理**：過去の服薬履歴を時系列で一覧表示し、残弾（総量）の管理も自動で行います。

## 🛠️ 技術スタック

- **言語 / UI**: Kotlin / Jetpack Compose
- **アーキテクチャ**: MVVM
- **データベース**: Room (SQLite)
- **バックグラウンド処理**: AlarmManager, NotificationManager, BroadcastReceiver
- **その他**: アダプティブアイコン対応

## 🚀 ビルドと実行方法

このプロジェクトには、ビルドとローカルホスティングを自動化するスクリプトが同梱されています。

1. **ビルドスクリプトの実行**
   ```powershell
   .\build.bat
   ```
   このスクリプトは以下の処理を自動で行います：
   - `JAVA_HOME` を自動設定して Gradle で APK をビルド
   - ビルド成功後、PCのローカルIPアドレスを検出
   - Pythonの `http.server` をバックグラウンドで起動し、ポート `8000` でAPKをホスティング

2. **アプリのインストール**
   コンソールに出力されたURL（例: `http://192.168.x.x:8000/app-debug.apk`）にスマートフォンのブラウザからアクセスし、APKをダウンロードしてインストールしてください。
